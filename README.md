## サイト概要
お酒好きのためのSNS

### サイトテーマ
お酒版のインスタグラム

### テーマを選んだ理由
お酒の種類が多く飲める量も限られているので、好みの合う人と繋がっておすすめのお酒を知れるようなSNSがあれば効率よくお酒を楽しめると思ったから。

### ターゲットユーザ
お酒好きな人

### 主な利用シーン
お酒を飲んだ時、飲みたい時
